# Polymarket Research & Documentation

This repository contains comprehensive documentation and research on Polymarket trading strategies, algorithms, and tools.

## Structure

- **`.cursor/rules/docapi/polymarket/`** - Complete Polymarket API documentation and trading strategies
  - `algorithms/` - Trading algorithms (sentiment analysis, on-chain tracking, market making, ML/NLP)
  - `tools/` - External tools and platforms
  - `websocket-api.md` - Real-time WebSocket API documentation
  - `limitations.md` - API limitations and best practices

## Content

### Trading Algorithms
- **Twitter Sentiment Analysis** - Social media sentiment tracking
- **On-Chain Analysis** - Whale tracking and blockchain monitoring
- **Market Making** - Liquidity provision strategies
- **ML/NLP** - Machine learning models for prediction

### Risk Management
- Portfolio optimization
- Position sizing
- Risk metrics and monitoring
